📋 Project Overview
This Portfolio Website is a personal professional website showcasing your skills, projects, certifications, and experience as a Web Developer and AI/ML enthusiast.

🛠️ Technology Stack
Category	Technologies
Frontend Framework	React.js (v19.1.1)
Language	JavaScript
Styling	HTML & CSS
Backend & Database	Firebase (Firestore)
Hosting	Firebase Hosting
Routing	React Router DOM (v7.9.2)
Icons	React Icons (v5.5.0)
HTTP Client	Axios (v1.12.2)
Email Service	EmailJS (v4.4.1)
Testing	Jest & React Testing Library
✨ Key Features
Responsive Design - Modern, mobile-friendly UI
Dynamic Content - Projects & certifications management
Admin Dashboard - Secure login to manage portfolio content:
Manage Home, About, Skills, Projects, Certifications
Add/edit/delete content without code changes
Contact Form - EmailJS integration for direct messaging
Skills Visualization - Display technical proficiencies with skill bars
Animations - Custom animations (login particles, background effects)
Multi-page SPA - Home, About, Skills, Projects, Certifications, Contact pages
🚀 What Makes It Different
Admin Control - Unlike static portfolios, you can manage content dynamically via dashboard
Firebase Backend - Real-time data updates without server maintenance
Scalable Architecture - Easy to add new sections or modify existing ones
Professional Animations - Particle effects and smooth transitions
Email Integration - Direct contact capability through EmailJS
Live & Deployed - Publicly accessible at https://srinivasapallapu9.web.app
📁 Project Structure
/src/pages - Main pages (Home, About, Skills, Projects, etc.)
/src/components - Reusable UI components (Navbar, Cards, Footer)
/src/admin - Dashboard for content management
/src/animations - Custom animation logic
/src/services - Firebase configuration & integration
/public - Static assets
This is a full-stack portfolio project combining frontend excellence with backend flexibility!
